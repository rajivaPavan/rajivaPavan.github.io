## Quick Start

```bash
# clone the repository
$ git clone https://github.com/rajivaPavan/rajivaPavan.github.io

# navigate to the repository
$ cd rajivaPavan.github.io

# install dependencies
$ npm install

# start development server
$ npm run dev
```
